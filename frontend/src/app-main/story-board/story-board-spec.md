

## Scene Tree Composition

Scene Components
- Enter Transition with Title
- Content with thematic music
- Leave Transition

## Types of Scene:
- Intro
- External Long Video Content (rare usage)
- Flow Sequence with Keyframe (Activity flow)
- Focus Box Scrollable transition (useful for map parts, collage, )
- Item Collection Sequence
- Vertical Comparison
- Single Shot (One after another)
- Face Time
- Outro

### Intro and Story layout
A good demonstrating video generally has following Scene flows
- Scene: A punching subject title intro
- Scene: Brand Logo Animation
- Scene: intent and dissection of the whole idea
- Scene: Pre description
- Scene: Main points
- Scene: Ending with follow up chain

###  Data Collection Sequence:

For collection, each data item consists of  `data-term` and `data-description`.
Data Term is image &/ title

**Two type of Presentation Flow**
- First Scene(Optional), Data term is presented in sequence
- Following Scene(Must), Data term and Description form a shot and then anticipation of one shot at a time

**Data Term presentation**
- Generally a grid/list is shown with data term
- Two type of style of Appearance:
  - in sequence and gets cleared all at once
  - one by one with attention graving effect

**Data Description along with term Presentation**
- Each data description is shown with proper text transition
- Text transition style: `Type writing`, `word-to-word writing`,
  `motion-writing`, '3D flip',  `blur-reveal`
- Number with text caption
- Number has counting transition with (odo sound)


### Single Item Shot scene
- Only one shot is shown at once with styled(static/looping) background
- a Proper padding/margin should be applied for focus consistency


### Vertical Data comparison
  - All terms are shown first
  - terms appearance style: slide-down, slide-up, flip-reveal
  - for multiple attribute comparison, each attribute appear one by one in sequence

### A-B roll:
- A-roll Facetime shot  with slow pan-in, pan-out. Face expression should be quick zoomed
- B-Roll Stock Video/interesting footage/expression video
  or the contextual video closely relevant to the topic

### Outro (Thank you so much, see you in next video)
