# playbook (frontend)

A project where ZeroMetry, LitSprit, VidLight MVPs are born

## Install the dependencies
```bash
yarn
# or
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```


### Build the app for production
```bash
quasar build
```

### Customize the configuration
See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).

## POC page

After starting the dev server, open the app and visit /poc to see the proof-of-concept page.

If you use pnpm (recommended in this workspace), run:

```powershell
pnpm install
pnpm dev
```

Then navigate to http://localhost:8080/poc (Quasar opens a browser automatically by default).
